**ApNv_cnidoglandular_SAMap.ipynb**: generating .h5ad files from each Astrangia and Nematostella dataset and running SAMap
**ApNv_compare_collapse.txt**: collapsing Astrangia and Nematostella cnidoglandular subclusters into broad clusters for SAMap comparison
**Apoc_embryo_scRNAseq_Seurat.txt**: analysis of whole Astrangia developmental cell atlas
**Apoc_embryo_scRNAseq_cnido_subset.txt**: generating cnidocyte subset from Astrangia scRNAseq data
**Apoc_neurogland_mucous_subset.txt**: generating neuroglandular subset from Astrangia scRNAseq data
**Apoc_whole_embryo_URD.txt**: generating URD tree from whole Astrangia developmental cell atlas
**Cellranger_mkref.txt**: making a reference genome in Cellranger to align scRNAseq reads
**Hydra_SAMap_input.txt**: extracting data from Hydra interstitial cell scRNAseq data to be used in SAMap
**SAMap_scR_input.txt**: extracting data from Astrangia and Nematostella cnidoglandular datasets to be used in SAMap
**dev_cnido_URD**: generating URD tree from Astrangia cnidocyte subset
**embryo_Cellranger_count.txt**: running Cellranger count with Astrangia developmental cell atlas data
**subcluster_genes_cells_numbers.txt**: extracting the total number of genes expressed and number of cells per each cluster in Astrangia and Nematostella cnidoglandular datasets
**xenia_cnido_subset.txt**: making a cnidocyte subset from the Xenia all (adult and regeneration) scRNAseq dataset
